# 14-Tech-Blog

## Description

This project is a tech blog. Its main purpose is to keep people updated on current tech talk.

## Usage

Once the application is deployed you will be able to navigate to the sign-up, login, or home page. To create an account navigate to the sign up page and fill out all fields, then click submit to create. To login in, use your credentials created to fill out the fields on the login page. The homepage will have the blog posts.
